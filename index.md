## Setting Bot
- First Go to `botConfig.js`
- Place Your Bot Token and Prefix
- 
```
module.exports = {
    token: "YOUR BOT TOKEN",
    prefix: "YOUR BOT PREFIX",
    admins: [
        "778802442942677064"
],
    debug: true,
    countChannel: "countChannelID"
};
```
